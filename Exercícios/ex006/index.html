Principais Formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito utilizando a tag <b > (não semântico)

Nesta outra frase, temos um termo em destaque utilizando a tag <strong> (semântico)

Itálico / Ênfase

Nesta frase, temos um termo em itálico utilizando a tag <i> (não semântico)

Nesta outra frase, temos um termo em ênfase utilizando a tag <em> (semântico)


OBS: o negrito e itálico não semântico, não são mais utilizados em HTML5, mas sim em CSS, por serem tags de forma

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>.

As cores do marcador podem ser mudadas com o parâmetro style de CSS utilizando um seletor na parte head, de forma que mudaria as cores de todas as marcações e não só de apenas uma.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Usa-se a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Usa-se a tag <ins>

A tag <u> é utilizada também para sublinhar, mas apenas como uma questão de estilo, logo está ligada às CSS

Texto sobrescrito

Para inserir coisas do tipo x2+y3, utiliza-se a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O, utiliza-se a tag <sub>
